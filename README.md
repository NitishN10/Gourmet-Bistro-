# Gourmet-Bistro-

A modern, responsive, and elegant digital menu interface designed for high-end dining establishments. Built with a clean, professional aesthetic to showcase culinary offerings.

Features
Dynamic Filtering: Users can filter the menu by category (Appetizers, Mains, Desserts, Drinks) using a smooth, animated interface.
Responsive Design: Fully optimized for mobile, tablet, and desktop viewing.
Elegant UI/UX: Features a sophisticated color palette, custom typography (Playfair Display & Inter), and subtle scroll-triggered animations.
Ready-to-Use Structure: Pre-built sections for featured items, special offers, and footer contact information.
Lightweight: Built using standard HTML5, CSS3, and vanilla JavaScript—no external frameworks or heavy dependencies required.

Technologies Used
HTML5: Semantic markup.
CSS3: Custom properties (variables), Flexbox, CSS Grid, and custom keyframe animations.
JavaScript: Vanilla JS for category filtering and Intersection Observer API for scroll animations.

Fonts/Icons: Google Fonts (Playfair Display, Inter) and Font Awesome 6.

How to Use
Clone the repository:
git clone [your-repository-url]

2.  **Open the project:** Open `index.html` in your web browser.
3.  **Customize:**
    *   **Content:** Modify the text and prices directly within the `index.html` file.
    *   **Images:** Replace the placeholder background URLs in the CSS header and item blocks with your own restaurant photography.
    *   **Styles:** Update the CSS variables in the `:root` section to match your brand's color scheme.
